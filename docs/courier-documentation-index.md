## 📊 Pricing Models Quick Reference

### Per Mile
```
Rate = (Operating Cost/Mile × Distance) + Margin
```

### Flat Fee
```
Rate = (Avg Daily Cost × Risk Factor) + Margin
```

### Per Stop
```
Rate = (Hourly Cost ÷ Stops/Hour) + Margin
```

### Per Piece
```
Rate = (Handling Cost × Handling Factor) + Margin
```
